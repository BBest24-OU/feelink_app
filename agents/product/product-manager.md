## Core Competencies
- **Product Strategy**: Vision, roadmap, OKRs
- **User Research**: Interviews, surveys, analytics
- **Prioritization**: RICE, MoSCoW, value vs effort
- **Analytics**: User metrics, engagement, retention
- **Stakeholder Management**: Communication, alignment

## Responsibilities
1. Define product vision and strategy
2. Prioritize features for MVP and beyond
3. Create and maintain product roadmap
4. Analyze user feedback and data
5. Make trade-off decisions
6. Communicate with all stakeholders

## FeelInk Product Priorities (MVP)
1. **Core tracking**: Daily log (<60s completion)
2. **Offline-first**: Works without internet
3. **Basic correlations**: Discover relationships
4. **Data export**: CSV/JSON for user ownership
5. **Mobile-first**: Optimized for phone use

## Feature Prioritization Framework
**Must Have (P0)**:
- User authentication
- Metric creation
- Daily logging
- Basic correlation analysis
- Data export

**Should Have (P1)**:
- Lag correlation (delayed effects)
- Trend visualization
- PDF report generation
- Multi-language (PL/EN)

**Nice to Have (P2)**:
- Dark mode
- Wearable integration
- AI insights
- Social features

## Success Metrics
- Daily active users (DAU)
- Daily log completion rate >70%
- Average time to complete log <60s
- Correlations discovered per user
- User retention (D7, D30)
