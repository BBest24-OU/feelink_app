## Logo Usage Guidelines

### âœ… DO

- Use official logo files only
- Maintain minimum clear space (150% of logo height)
- Respect minimum sizes (32px digital, 15mm print)
- Use approved color variations only
- Place on solid, high-contrast backgrounds

### âŒ DON'T

- Stretch, distort, or rotate the logo
- Change colors outside brand palette
- Add effects (shadows, glows, gradients)
- Place on busy or low-contrast backgrounds
- Recreate or modify the logo

## Color Variations

### 1. Primary (Teal on White)
- **Use for:** Most applications
- **Background:** White (#FFFFFF)
- **Logo:** Teal (#14B8A6)

### 2. Reverse (White on Teal)
- **Use for:** Dark backgrounds
- **Background:** Teal (#14B8A6) or Slate (#1E293B)
- **Logo:** White (#FFFFFF)

### 3. Monochrome (Slate on White)
- **Use for:** Professional contexts, print
- **Background:** White (#FFFFFF)
- **Logo:** Slate (#1E293B)

### 4. Grayscale
- **Use for:** Black & white printing only
- **Background:** White
- **Logo:** Black (#000000)

## Minimum Sizes

### Digital
- **Minimum height:** 32 pixels
- **Recommended:** 48px+ for clarity
- **Mobile app icon:** 192px or 512px

### Print
- **Minimum height:** 15mm
- **Recommended:** 20mm+ for clarity
- **Business card:** 25mm+

---

## File Formats

### Vector (Preferred)
- **SVG:** Web, digital, scalable
- **AI/EPS:** Print, professional design

### Raster (When Needed)
- **PNG:** Web, transparency needed
- **JPG:** Photography, large images
- **ICO:** Favicons only

**Always start with vector, export to raster as needed.**

## Brand Assets Checklist

Before finalizing logo for production:

- [ ] Logo works in monochrome
- [ ] Logo is legible at 16px (favicon size)
- [ ] Logo looks good reversed (white on color)
- [ ] Logo has appropriate clear space
- [ ] SVG file is optimized and clean
- [ ] PNG files have transparent backgrounds
- [ ] All colors use exact brand hex values
- [ ] File naming follows convention
- [ ] Accessibility tested (contrast ratios)
