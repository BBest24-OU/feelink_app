## Logo Concepts

### 1. The Link (RECOMMENDED) ⭐

**File:** `feelink-logo-concept-the-link.svg`

**Description:**
- Minimalist design featuring two abstract shapes connecting
- **Left shape**: Organic, fluid blob (representing feelings/emotions)
- **Right shape**: Geometric square with rounded corners (representing data/analysis)
- **Connection**: Subtle dot pattern linking the two shapes
- **Typography**: "FeelInk" in Inter font, clean and professional

**Visual Metaphor:** Bridging subjective feelings with objective insights

**Why it works:**
- Immediately communicates the app's purpose
- Simple enough to work at small sizes
- Distinctive and memorable
- Scalable from favicon to billboard

**Usage:**
- Primary logo for all applications
- App icon (simplified, icon-only version)
- Website header
- Marketing materials

---

### 2. The Pulse

**Description:**
- Abstract heart rate/mood curve forming letter "f"
- Smooth, calming wave that transitions from organic to structured
- Custom "i" dotted with small circle

**Visual Metaphor:** Tracking emotional patterns over time

**Best for:** Marketing materials emphasizing tracking/monitoring

---

### 3. The Connection Nodes

**Description:**
- Three dots arranged in triangle, connected by subtle lines
- Represents: tracking → correlation → insight
- Minimalist, clean, geometric

**Visual Metaphor:** Connecting data points to discover patterns

**Best for:** Technical documentation, developer materials

---

### 4. The Journey Path

**Description:**
- Abstract path/timeline with subtle elevation changes
- Represents personal mental health journey
- Single continuous line forming subtle "F" shape

**Visual Metaphor:** The ups and downs mapped over time

**Best for:** Onboarding screens, progress visualization

---

### 5. The Infinity Link

**Description:**
- Stylized infinity symbol (∞) with one loop organic, one geometric
- Represents continuous self-discovery cycle
- Subtle break in center forming two "i" dots

**Visual Metaphor:** Ongoing journey of self-understanding

**Best for:** Premium/professional version, therapy partnerships

## Logo Usage Guidelines

### ✅ DO

- Use official logo files only
- Maintain minimum clear space (150% of logo height)
- Respect minimum sizes (32px digital, 15mm print)
- Use approved color variations only
- Place on solid, high-contrast backgrounds

### ❌ DON'T

- Stretch, distort, or rotate the logo
- Change colors outside brand palette
- Add effects (shadows, glows, gradients)
- Place on busy or low-contrast backgrounds
- Recreate or modify the logo

---

## Color Variations

### 1. Primary (Teal on White)
- **Use for:** Most applications
- **Background:** White (#FFFFFF)
- **Logo:** Teal (#14B8A6)

### 2. Reverse (White on Teal)
- **Use for:** Dark backgrounds
- **Background:** Teal (#14B8A6) or Slate (#1E293B)
- **Logo:** White (#FFFFFF)

### 3. Monochrome (Slate on White)
- **Use for:** Professional contexts, print
- **Background:** White (#FFFFFF)
- **Logo:** Slate (#1E293B)

### 4. Grayscale
- **Use for:** Black & white printing only
- **Background:** White
- **Logo:** Black (#000000)

---

## Clear Space

Minimum clear space around logo = **1.5× logo height** on all sides

```
┌─────────────────────────┐
│                         │
│    ┌─────────────┐      │  ← 1.5h
│    │             │      │
│    │    LOGO     │  h   │
│    │             │      │
│    └─────────────┘      │  ← 1.5h
│                         │
└─────────────────────────┘
      ↑           ↑
     1.5h       1.5h
```

## Minimum Sizes

### Digital
- **Minimum height:** 32 pixels
- **Recommended:** 48px+ for clarity
- **Mobile app icon:** 192px or 512px

### Print
- **Minimum height:** 15mm
- **Recommended:** 20mm+ for clarity
- **Business card:** 25mm+

---

## File Formats

### Vector (Preferred)
- **SVG:** Web, digital, scalable
- **AI/EPS:** Print, professional design

### Raster (When Needed)
- **PNG:** Web, transparency needed
- **JPG:** Photography, large images
- **ICO:** Favicons only

**Always start with vector, export to raster as needed.**

### Design Requirements
- **Background:** Teal (#14B8A6)
- **Symbol:** White, simplified version of logo
- **Safe area:** 10% padding from edges
- **Border radius:** 20% (handled by OS)
- **No text:** Symbol only (readable at small sizes)

## Brand Assets Checklist

Before finalizing logo for production:

- [ ] Logo works in monochrome
- [ ] Logo is legible at 16px (favicon size)
- [ ] Logo looks good reversed (white on color)
- [ ] Logo has appropriate clear space
- [ ] SVG file is optimized and clean
- [ ] PNG files have transparent backgrounds
- [ ] All colors use exact brand hex values
- [ ] File naming follows convention
- [ ] Accessibility tested (contrast ratios)
